Cara setup Google OAuth
  1. Google Cloud Console → enable YouTube Data API v3
  2. Create OAuth Client ID (Web) → tambah Redirect URI di atas

Cara dapat API Key
  1. Daftar di TikTok Shop Partner Center
  2. Buat App → dapatkan App Key & App Secret
  3. Authorize shop → dapatkan Access Token & Shop Cipher
  4. Simpan di form ini → klik Sync GMV

Export dari TikTok Shop → Affiliate Center → Analytics → Export CSV

Paste dari Excel/Sheets. Format per baris:

video_id, gmv, komisi
